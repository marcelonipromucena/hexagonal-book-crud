# Projeto de exemplo do design pattern Hexagonal

### Algumas observações
- A distribuição das classes e interfaces desse projeto foi feito através de módulos.
- O projeto utiliza das seguintes tecnologias
    - H2
    - Spring Boot
    - MapStruct
    - Lombok
    - Spring Data JPA
- O objetivo do projeto é realizar as operações de um crud simples seguindo o fluxo da arquitetura hexagonal.
- Para rodar o projeto basta
    - Apartir da pasta root rodar: mvn clean install
    - Entrar na pasta ms-launcher e rodar: mvn spring-boot:run
